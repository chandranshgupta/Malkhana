Legal Framework
Built for Indian forensic law

Every feature maps directly to a statutory obligation. Compliance is not a checkbox — it is the architecture.

BSA 2023 — §63
Section 63 Cert
Auto-generated certificate attesting device health, automated storage, integrity, and chain of custody. Replaces IEA §65B.
BSA 2023 — §63(2)(c)
Downtime Logging
System interruptions are auto-flagged in system_health_log. Certificate generation checks and discloses any downtime events.
BNSS 2023 — §153
Form CC-1
Mandatory chain-of-custody form auto-populated at seizure. Panch witness fields, Imaging Specialist, and Malkhana Safe-Keeper signatures enforced.
CERT-In / IT Act
180-Day Logs
Tamper-proof Merkle-chained audit log preserved for minimum 180 days. Any stealth edit breaks the Merkle root verification.
Triple-Hash Verification Protocol — Eliminating Malkhana Gaps
H1
Birth Hash
At crime scene seizure
SHA-256 + MD5 generated by IO. Baseline proof of original device state.
H2
Receipt Hash
Before opening seal at FSL
Malkhana In-Charge verifies. H1 ≠ H2 triggers a Malkhana Gap alert — grounds for evidence exclusion.
H3
Analysis Hash
After forensic imaging
FSL Examiner confirms post-extraction integrity. H2 ≠ H3 flags FSL alteration.
Technical Stack
Rust-native. Zero cloud surface.
Backend Runtime
Rust + Tauri v2
10× faster hashing than Python, memory-safe. ~5MB shell vs 150MB Qt alternative.
Frontend
React 19 + Tailwind
Blueprint/brutalist design language. Mono system-wide, uppercase labels, brutalist offset shadows.
Database
SQLCipher AES-256
rusqlite + bundled-sqlcipher. WAL mode. PBKDF2 256,000 iteration key derivation. Full at-rest encryption.
Audit Chain
Merkle Tree Log
SHA-256(prev_hash + entity + actor + details) on every log entry. Any manual edit fails chain verification.
Cryptographic Signing
Ed25519-dalek
Certificate sealing and session-as-custody events. Biometric co-sign workflow with dual-authentication.
Multilingual
22 Indian Languages
All scheduled languages treated equally via i18next. RTL support for Urdu, Sindhi, Kashmiri. No language hierarchy.
